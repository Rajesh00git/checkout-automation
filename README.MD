# 🛒 Checkout Automation - QA-1234

E-commerce checkout flow automation using Pytest + Playwright with CI/CD integration.

## 📋 Project Overview

Automated smoke tests for checkout flow covering:
- Guest user checkout with Cash on Delivery (COD)
- Logged-in user purchase with saved payment
- Cross-browser testing (Chrome + Firefox)

## 🚀 Tech Stack

- **Framework:** Pytest 9.0.2
- **Browser Automation:** Playwright 1.43.0
- **Design Pattern:** Page Object Model (POM)
- **CI/CD:** GitHub Actions
- **Reports:** HTML reports with screenshots
